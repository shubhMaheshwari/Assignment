Front End for the User Directory Application

Table of Contents

The front end for the user directory application is realized in series of steps. The steps are:

1 Client Side Routing as a State Machine

The front end application is visualized as a DFA state machine with a set of states ans transitions. The transitions are the events and transition to each state is accomplished using client side routing.

2 Separating View and Model

For the same user directory application, the model is separated from the view.

3 Model View Controller Pattern

The design principle of model view controller is applied to the user directory application.

4 Controller as Reactive

The controller implemented in the user directory application reacts to the outcome of the operation on model via success and failure handlers.
